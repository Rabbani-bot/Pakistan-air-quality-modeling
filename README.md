# 🌫️ Spatial-Temporal Dynamics of Pakistan's Air Quality (2015-2030)

## Overview
This repository contains a localized panel dataset and a Python-based execution environment (Jupyter Notebook) for tracking, analyzing, and forecasting $PM_{2.5}$ atmospheric pollution across Pakistan's major industrial hubs.

## Methodological Approach
1. **Geospatial Analytics:** Mapping the "Punjab Smog Belt" using 2025 cross-sectional averages.
2. **Seasonal Atmospheric Multipliers:** Calculating the impact of thermal inversion during peak winter months vs. baseline summer minimums.
3. **Forecasting:** Utilizing an Ordinary Least Squares (OLS) Linear Regression model to project baseline atmospheric trajectories through 2030 against the WHO $5.0 \ \mu g/m^3$ safe exposure threshold.

## Tech Stack
* **Language:** Python 3.10+
* **Libraries:** `pandas` (Data Manipulation), `matplotlib` & `seaborn` (Data Visualization), `scikit-learn` (Predictive Modeling).
